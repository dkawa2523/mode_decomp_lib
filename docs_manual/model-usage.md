# 用途ごとのモデルの説明

## モード分解 / 前処理 / 学習 / 評価の整理
| 区分 | 目的 | 入力 | 出力 | 代表的な成果物 |
| --- | --- | --- | --- | --- |
| モード分解 | 空間分布を低次元表現へ変換 | 分布データ | 係数 | 再構成誤差、係数分布、空間比較図 |
| 前処理 | 係数の正規化・次元調整 | 係数テーブル | 前処理係数 | 分布・相関・スペクトル図 |
| 学習 | 条件→係数の写像学習 | 条件 + 係数 | 学習済みモデル | 学習曲線、予測 vs 真値 |
| 評価 | 精度指標の算出 | 真値と推定 | 指標 | RMSE/MAE/R2 等 |

## モデル用途分類（代表）
| 用途 | 推奨モデル | 特徴 | 主なライブラリ |
| --- | --- | --- | --- |
| 軽量・高速学習 | Ridge / Lasso | 高速、安定 | scikit-learn |
| 非線形回帰 | XGBoost / LightGBM / CatBoost | 非線形・高精度 | xgboost / lightgbm / catboost |
| 不確かさ推定 | MTGP / GPR | 分散推定、少量データ向け | gpytorch / scikit-learn |
