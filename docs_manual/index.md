# 概要

## 想定背景と課題
- 複数条件の空間分布データを扱い、条件→分布の関係を効率よく学習・推論したい。
- モード分解による次元削減、前処理、学習、推論を一貫して実行し、結果を可視化・比較できる必要がある。
- 出力や処理単位が分かりやすく、開発・拡張が容易な構成が求められる。

## 本コードの概要
本コードは、空間分布データに対して**モード分解**を行い、係数を前処理・学習・推論へつなげるための一連のパイプラインを提供します。入力データは条件CSVと各条件に対応する分布CSV（スカラー/ベクトル）で構成されます。出力は runs/<name>/<process> 配下に整理され、可視化・評価・モデル・ログが一貫した構成で保存されます。

## 機能一覧（要約）
- モード分解（解析関数系 / データ駆動系）
- 前処理（正規化 / PCA 等、手法依存ルール対応）
- 学習（複数モデルの学習・評価）
- 推論（単一 / バッチ / 最適化）
- 画像・表の可視化出力
- leaderboad（分解法 / 学習法 比較）
- ClearML 連携（任意）
- データセット管理（CSV/NPY 両対応）

## 本コードの有用性
- 分解→学習→推論の一貫運用により、実験の再現性と比較性を確保。
- ドメイン別（矩形 / 円 / 環 / 任意マスク / 球面）に適した分解法を選択可能。
- 出力構成が固定のため、結果の探索や自動集計が容易。

## 用語統一
- **特長量化**はすべて**モード分解**として表記します。
